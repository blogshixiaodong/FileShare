<b>JavaWeb: 文件共享系统</b>

项目介绍：
主要使用JSP+Servlet简单实现了三个层次，JSP负责页面的显示，Servlet负责逻辑的控制，Service是业务流程的处理，DAO数据库访问层。主要实现了系统的登陆功能，文件的上传下载，留言板功能

PS：
学习JavaWeb一段时间的简单应用，准备接下来学习一些框架技术，在实现系统时总是觉得代码耦合度很高

<b>开发环境</b>
MyEclipse+Tomcat+MySql
前端框架：Bootstrap，JQuery


<b>功能介绍</b>
<ul>
       <li>用户注册</li>
       <li>用户登陆</li>
       <li>文件上传</li>
       <li>文件下载</li>
       <li>用户留言</li>
</ul>
